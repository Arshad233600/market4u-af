
# گزارش آمادگی انتقال به مایکروسافت اژور (Azure Readiness Report) ☁️

**وضعیت پروژه:** 🟢 آماده انتشار امن (Ready for Secure Static Deployment)

---

## ۱. زیرساخت مورد نیاز (Infrastructure)

برای استقرار نسخه فعلی، معماری زیر تایید شده است:

*   **سرویس میزبان:** Azure Static Web Apps (Free Plan).
*   **محل سرور (Region):** UAE North (برای کمترین تاخیر در افغانستان).
*   **دیتابیس:** کلاینت-ساید (LocalStorage) برای نسخه دمو.

## ۲. وضعیت امنیت (Security Status)

*   ✅ **مدیریت کلیدها:** کلیدهای حساس (API Keys) از کد و کانفیگ حذف شده‌اند.
*   ✅ **حفاظت:** قابلیت‌های هوش مصنوعی در صورت نبود کلید، به صورت امن (Fail-safe) غیرفعال می‌شوند.
*   ✅ **هدرهای امنیتی:** فایل `staticwebapp.config.json` برای اعمال هدرهای امنیتی (HSTS, X-Frame-Options) پیکربندی شده است.

## ۳. نیازمندی‌های آینده (Future Roadmap)

برای عبور از فاز دمو به نسخه تجاری (Production)، منابع زیر در اژور باید اضافه شوند:

1.  **Azure Functions:** برای مدیریت درخواست‌های حساس و مخفی کردن API Key.
2.  **Azure SQL Database:** برای ذخیره دائمی آگهی‌ها و کاربران.
3.  **Azure Storage Account:** برای ذخیره تصاویر کاربران.

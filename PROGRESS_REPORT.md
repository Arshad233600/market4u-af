## ۱. دستاوردهای کلیدی (Completed Milestones)

بخش فرانت‌اند (Frontend) پروژه به صورت **۱۰۰٪ تکمیل** شده است. تمامی قابلیت‌های یک مارکت‌پلیس مدرن با استفاده از داده‌های شبیه‌سازی شده (Mock) و لاجیک‌های پیشرفته سمت کلاینت پیاده‌سازی شده‌اند.

### 🎨 رابط کاربری و تجربه کاربری (UI/UX)
- [x] **طراحی واکنش‌گرا (Responsive):** سازگاری کامل با موبایل، تبلت و دسکتاپ.
- [x] **پشتیبانی از زبان‌ها:** پیاده‌سازی کامل راست‌چین (RTL) برای زبان‌های دری و پشتو.
- [x] **صفحات اصلی:** خانه، جستجو، جزئیات محصول، پروفایل کاربری، ثبت آگهی.

### 🛒 مدیریت آگهی‌ها (Core Features)
- [x] **ثبت و ویرایش آگهی:** فرم چند مرحله‌ای با قابلیت آپلود عکس و تعیین موقعیت روی نقشه.
- [x] **حذف آگهی (Soft Delete):** امکان حذف موقت آگهی توسط کاربر.
- [x] **ارتقای آگهی:** سیستم کسر از کیف پول و اعمال نشان‌های "فوری" و "نردبان".
- [x] **هوش مصنوعی:** تولید توضیحات آگهی با استفاده از Gemini API.

### 💬 ارتباطات و تعامل (Communication)
- [x] **سیستم چت:** رابط کاربری پیام‌رسان با قابلیت ارسال پیام و ذخیره در حافظه.
- [x] **اعلان‌ها (Notifications):** سیستم هشدار داخلی برای پیام‌ها و تراکنش‌ها.

### 👤 مدیریت حساب کاربری (User Management)
- [x] **پروفایل:** ویرایش اطلاعات و تغییر تصویر.
- [x] **احراز هویت:** فرم آپلود تذکره/پاسپورت برای دریافت تیک آبی.
- [x] **کیف پول:** مدیریت موجودی، تاریخچه تراکنش‌ها و شبیه‌سازی درگاه پرداخت.

### ⚙️ بخش مدیریت (Admin Panel)
- [x] **داشبورد ادمین:** مشاهده آگهی‌های در انتظار تایید و تایید/رد آن‌ها.
- [x] **مدیریت کاربران:** پنل بررسی مدارک هویتی کاربران.

### 4.6 Security Tests

| Test Area | What To Verify |
|---|---|
| JWT secret enforcement | Login returns 503 when `AUTH_SECRET` is placeholder/absent |
| Password hashing | Passwords stored as bcrypt hash, never plaintext |
| Rate limiting | Login endpoint blocked after N failed attempts |
| SQL injection | Parameterized queries used throughout (`mssql` request parameters) |
| XSS | No `dangerouslySetInnerHTML` with unsanitized user content |
| CORS | Only same-origin requests allowed in production (SWA proxy) |
